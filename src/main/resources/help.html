Aide

Écran de sélection des fichiers

  • Ctrl+O : Ouvrir un fichier Excel contenant des navettes de facturation
  • Ctrl+L : Charger une saisie sauvegardée ainsi que le paramétrage lors de l'enregistrement
  • Ctrl+M : Charger une saisie sauvegardée en conservant le paramétrage actuel. Attention : des saisies qui étaient valides pourraient ne plus l'être
  • Ctrl+Droite : Suivant
  • Ctrl+S : Sauvegarder la saisie

Écran de saisie des données

  • Ctrl+S : Sauvegarder la saisie
  • Ctrl+Droite : Suivant
  • Ctrl+Gauche : Précédent

Écran de prévisualisation et écriture du fichier de sortie

  • Ctrl+Droite : Suivant, écrire le fichier de facturation
  • Ctrl+Gauche : Précédent, retour sur l'écran de saisie
  • Ctrl+O : Ouvrir le fichier de facturation
  • Ctrl+D : Ouvrir le dossier de facturation

* Si le fichier existe déjà, il sera écrasé.

** Si le fichier existant est ouvert, l'écriture échouera.

*** Si le fichier existe déjà, le fait de l'ouvrir n'effectuera pas la mise à jour de celui-ci. Un texte de confirmation d'écriture du fichier sera écrit en bas de l'écran une fois l'action réalisée.

Astuce : Il est possible d'enregistrer un environnement paramétré avec une saisie vide.
